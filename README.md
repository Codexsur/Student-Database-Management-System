# Student Database Management System

## Overview
This project is a simple Student Database Management System developed using SQL and MySQL.

## Features
- Add student records
- Update student information
- Delete records
- Search and filter students
- Generate department-wise reports

## Technologies Used
- SQL
- MySQL

## Concepts Used
- CRUD Operations
- GROUP BY
- Aggregate Functions
- Joins & Subqueries
